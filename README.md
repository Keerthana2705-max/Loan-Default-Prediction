  <h1>Loan Default Prediction System</h1>
        <p>Machine Learning Project using Python</p>
        <h2>Introduction</h2>
        <p>
            The <span class="highlight">Loan Default Prediction System</span> 
            is a machine learning project developed to analyze borrower data 
            and predict loan repayment risks. The system helps financial 
            institutions identify risky borrowers and make better lending decisions.
        </p>
        <h2>Objectives</h2>
        <ul>
            <li>Analyze borrower loan data</li>
            <li>Predict loan default risk</li>
            <li>Perform data preprocessing and visualization</li>
            <li>Build a machine learning prediction model</li>
            <li>Create interactive dashboards</li>
        </ul>

        <h2> Key Insights</h2>
<ul>
            <li>Higher interest rates increase default risk.</li>
            <li>Low-income borrowers default more frequently.</li>
            <li>Employment length affects repayment behavior.</li>
            <li>Debt consolidation loans are common.</li>
            <li>Risk classification improves decision-making.</li>
        </ul>
